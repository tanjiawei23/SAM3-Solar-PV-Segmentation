# SAM3-Solar-PV-Segmentation

## 🚀 Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/tanjiawei23/SAM3-Solar-PV-Segmentation.git
   cd SAM3-Solar-PV-Segmentation
   
1. Install the required dependencies:
   ```bash
   pip install ultralytics numpy Pillow

2. Download the pre-trained SAM 3 weights (sam3.pt) from the official Ultralytics release and place it in the root directory.
